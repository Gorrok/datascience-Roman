## Алгоритм работы агента

### При получении запроса:
1. **Анализ контекста** - определение типа задачи
2. **Поиск в базе данных** - проверка решенных задач
3. **Навигация по папкам** - поиск в соответствующих категориях
4. **Создание решения** - использование шаблонов и лучших практик
5. **Документирование** - добавление в базу данных

### При сложных проблемах:
1. **Поиск в базе данных** - проверка существующих решений
2. **Поиск в Reddit** - `site:reddit.com/r/n8n [ключевые слова]`
3. **Поиск в форуме** - `site:community.n8n.io [ключевые слова]`
4. **Создание нового решения** - документирование в базе данных
