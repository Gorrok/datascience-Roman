# n8n Workflows - Главная папка

## Описание
Основная папка для хранения всех рабочих процессов n8n. Здесь организованы процессы по категориям для удобной навигации и управления.

## Структура папок

### 📁 [automation/](./automation/)
Автоматизированные процессы для рутинных задач
- **Путь**: `./automation/`
- **Описание**: Процессы автоматизации повседневных задач
- **Примеры**: Уведомления, обработка данных, синхронизация

### 📁 [integrations/](./integrations/)
Интеграции с внешними сервисами и API
- **Путь**: `./integrations/`
- **Описание**: Процессы для подключения к внешним сервисам
- **Примеры**: CRM, email-сервисы, социальные сети

### 📁 [data_processing/](./data_processing/)
Обработка и трансформация данных
- **Путь**: `./data_processing/`
- **Описание**: Процессы для работы с данными
- **Примеры**: ETL, валидация, форматирование

### 📁 [templates/](./templates/)
Готовые шаблоны процессов
- **Путь**: `./templates/`
- **Описание**: Переиспользуемые шаблоны
- **Примеры**: Базовые структуры, примеры узлов

### 📁 [monitoring/](./monitoring/)
Мониторинг и логирование
- **Путь**: `./monitoring/`
- **Описание**: Процессы для отслеживания работы системы
- **Примеры**: Алерты, отчеты, метрики

### 📁 [documentation/](./documentation/)
Документация и справочные материалы
- **Путь**: `./documentation/`
- **Описание**: Техническая документация и гайды
- **Примеры**: API документация, лучшие практики

## Быстрая навигация

### По типу процесса:
- **Автоматизация** → `./automation/`
- **Интеграции** → `./integrations/`
- **Обработка данных** → `./data_processing/`
- **Шаблоны** → `./templates/`
- **Мониторинг** → `./monitoring/`

### По сложности:
- **Простые** → `./templates/basic/`
- **Средние** → `./automation/`, `./integrations/`
- **Сложные** → `./data_processing/`, `./monitoring/`
