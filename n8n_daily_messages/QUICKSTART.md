# ⚡ Быстрый старт - n8n Daily Messages

## 🚀 За 2 минуты:

### 1. Подготовка (30 сек)
```
@BotFather → /newbot → сохранить токен
@username_to_id_bot → получить ID группы
Добавить бота в группу
```

### 2. Импорт в n8n (30 сек)
```
n8n → Import from File → daily_telegram_message.json
```

### 3. Настройка (1 мин)
```
Credentials → New → Telegram API → вставить токен
Telegram Node → Chat ID: -1001234567890
Schedule Trigger → Cron: 0 9 * * * (9:00 каждый день)
Activate Workflow ✅
```

## 📋 Частые cron выражения:

```bash
0 9 * * *       # Каждый день в 9:00
0 18 * * *      # Каждый день в 18:00
0 9 * * 1-5     # Будни в 9:00
0 9 * * 1       # Каждый понедельник в 9:00
0 9,18 * * *    # Каждый день в 9:00 и 18:00
```

## 🔧 Решение проблем:

**Сообщение не отправляется:**
- Проверьте Chat ID (начинается с `-100`)
- Убедитесь что бот в группе
- Проверьте токен в credentials

**Неправильное время:**
- Проверьте timezone в Schedule Trigger
- Используйте [crontab.guru](https://crontab.guru/)
