# Data Science Projects Portfolio

Репозиторий с проектами по Data Science, автоматизации и разработке.

## 📁 Структура проекта

```
datascience-Roman/
├── portfolio/              # Портфолио проектов (кейсы для работодателя)
├── telegram_member_tracker/  # Telegram бот для отслеживания участников каналов
├── superagent/             # Система автоматизации процессов n8n
├── personal_agent/         # AI-агент для анализа рыболовных данных
├── scripts/               # Утилиты и скрипты (.sh, .exp, .py)
├── sql/                   # SQL запросы
├── js/                    # JavaScript скрипты
└── json_workflows/        # JSON конфигурации и workflows
```

## 🚀 Проекты

### [📊 Портфолио](./portfolio/)
Краткие кейсы реализованных проектов:
- **Telegram Member Tracker** - автоматизация мониторинга участников каналов
- **n8n Superagent** - система автоматизации процессов

### [🤖 Telegram Member Tracker](./telegram_member_tracker/)
Бот для отслеживания вступлений и выходов участников в Telegram каналах с сохранением данных в Google Sheets.

### [⚙️ n8n Superagent](./superagent/)
Комплексная система для работы с n8n процессами: организованная структура, документация, шаблоны и база знаний.

## 🛠️ Технологии

- **Python** - основной язык разработки
- **Telegram Bot API** - разработка ботов
- **Google Sheets API** - интеграция с таблицами
- **n8n** - автоматизация процессов
- **SQLAlchemy** - работа с базами данных
