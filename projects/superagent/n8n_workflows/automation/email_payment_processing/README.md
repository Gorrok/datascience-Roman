# Рабочий процесс: Обработка данных из вложенных файлов

Этот рабочий процесс n8n автоматизирует извлечение данных из файлов (PDF, TXT), прикрепленных к электронным письмам, и сохраняет их в Google Docs. Рабочий процесс также может выполнять разные действия в зависимости от фамилии исполнителя, найденной в документе.

## Как это работает

1.  **Триггер (IMAP Email)**: Рабочий процесс запускается при получении нового письма и автоматически скачивает вложенные файлы.

2.  **Проверка наличия вложений (If Node)**: Процесс продолжается, только если в письме есть хотя бы один вложенный файл.

3.  **Извлечение текста из файла (Function Node)**: Этот узел извлекает текст из вложения.
    * s   **Для `.txt` файлов**: Текст читается напрямую.
    *   **Для `.pdf` файлов**: Требуется дополнительная настройка или внешний сервис для извлечения текста, так как n8n не имеет встроенной функции чтения PDF. В шаблоне это место обозначено комментарием.

4.  **Извлечение данных (Function Node)**: С помощью JavaScript и регулярных выражений из извлеченного текста находятся нужные данные (например, ФИО, сумма, дата).

5.  **Ветвление по исполнителю (Switch Node)**: В зависимости от найденной фамилии, рабочий процесс может пойти по разным веткам для выполнения специфических действий.

6.  **Запись в Google Docs (Google Docs Node)**: Извлеченные данные форматируются и добавляются в указанный документ Google Docs.

## Настройка

1.  **Импортируйте workflow**: Загрузите файл `workflow.json` в ваш n8n.
2.  **Настройте триггер**: В узле "IMAP Email Trigger" укажите данные для подключения к вашему почтовому ящику.
3.  **Настройте извлечение текста из PDF (при необходимости)**: Если вы работаете с PDF, в узле "Extract Text from Attachment" вам нужно будет добавить логику для их обработки (например, через API внешнего сервиса).
4.  **Настройте извлечение данных**: Адаптируйте код в узле "Parse Data from Text" под структуру ваших документов.
5.  **Настройте ветвление**: В узле "Route by Employee" укажите фамилии и настройте нужные ветки.
6.  **Настройте Google Docs**: В узле "Append to Google Doc" пройдите аутентификацию, укажите ID вашего документа.
7.  **Активируйте workflow**.
