# ZAZ - Агрегатор данных из Google Sheets

Этот workflow предназначен для сбора и агрегации данных из трех различных Google Таблиц в одну итоговую таблицу.

## Задача

Автоматически собирать данные из трех источников и объединять их по заданной логике для дальнейшего анализа.

## Логика работы (Предварительная)

1.  **Чтение данных**: Workflow последовательно подключается к трем указанным Google Таблицам и забирает из них данные.
2.  **Объединение**: Данные из трех источников объединяются в один поток.
3.  **Запись результата**: Итоговый набор данных записывается в целевую Google Таблицу.

**Примечание:** Текущая логика является предварительной. Для полноценной настройки workflow требуется уточнение деталей по структуре исходных таблиц и правил агрегации данных.

## Необходимые учетные данные (Credentials)

Для работы workflow необходим доступ к Google API с правами на чтение и запись в Google Sheets. Убедитесь, что в вашем n8n-инстансе настроены соответствующие `Google Sheets API` credentials.
