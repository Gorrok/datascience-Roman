# Telegram Member Tracker Bot

Разработал Telegram бот на Python для автоматического отслеживания вступлений и выходов участников в каналах с сохранением данных в Google Sheets. Реализовал интеграцию с Telegram Bot API и Google Sheets API, асинхронную обработку событий и логирование активности. Развернул решение на Linux сервере с автозапуском. Результат: полная автоматизация мониторинга участников и структурированные данные для аналитики.

**Технологии:** Python, Telegram Bot API, Google Sheets API, asyncio, SQLite
