# Мое портфолио: Автоматизация, n8n и боты

Привет! Здесь собраны краткие описания моих основных проектов. Я занимаюсь тем, что проектирую логику автоматизаций, настраиваю сложные сценарии и пишу скрипты (с активной помощью AI), чтобы "подружить" между собой разные сервисы. И сам же всё это разворачиваю на серверах.

## 🚀 Проекты

### 1. [Telegram Member Tracker Bot](../telegram_member_tracker/)

**Что это:** Бот для отслеживания, кто вступает в Telegram-каналы и откуда.
**Что я сделал:**

- Продумал архитектуру и с помощью AI написал бота на Python (aiogram, asyncio).
- Настроил связку Telegram API и Google Sheets API, чтобы данные автоматически улетали в табличку для аналитиков.
- Развернул бота на Linux-сервере (Ubuntu) и настроил автозапуск через systemd, чтобы бот не падал после перезагрузки сервера.

### 2. [n8n Superagent & AI Транскрибация](../superagent/)

**Что это:** Большая система автоматизации рабочих процессов, речевая аналитика и автоматический контроль качества.
**Что я сделал:**

- **Автоматизация Отдела Контроля Качества (ОКК):** Собрал в n8n сложную систему валидации закрытых сделок. Бот автоматически собирает данные (задачи, заметки, эксперты), прогоняет через кастомный движок валидации и мгновенно отправляет алерты о нарушениях в Telegram.
- **Речевая аналитика и транскрибация звонков (AI):** Спроектировал процесс автоматической расшифровки звонков (связка n8n + REST API AssemblyAI). Настроил обработку аудио, взаимодействие с нейросетью и автоматическую выгрузку готовых текстов в Google Таблицы для аналитиков.
- Заменил кучу разрозненных ручных действий на стабильные сценарии (workflow) в n8n с настроенной обработкой ошибок.

### 3. CRM для отдела маркетинга (Букмекерская компания)

**Что это:** Настройка amoCRM с нуля для специфических нужд маркетинга.
**Что я сделал:**

- Полностью внедрил amoCRM, настроил сложные автоматизации через триггеры F5.
- Настроил захват ID пользователей и источников трафика, чтобы понимать, откуда приходят клиенты.
- Собрал удобные дашборды в Looker Studio для руководства.

### 4. Чат-боты для музыкального лейбла

**Что это:** Снижение нагрузки на техподдержку.
**Что я сделал:**

- Собрал ботов на платформе Salebot.
- Автоматизировал процесс регистрации пользователей и выставления счетов (биллинг).
- Настроил умное распределение диалогов: простые вопросы бот закрывает сам, сложные — переводит на свободного оператора.
