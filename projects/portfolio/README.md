# Портфолио проектов

Краткое описание реализованных проектов для резюме.

## Проекты

### 1. [Telegram Member Tracker Bot](./01_telegram_member_tracker.md)
Разработал Telegram бот для автоматического отслеживания участников каналов с интеграцией Google Sheets.

### 2. [n8n Superagent](./02_n8n_superagent.md)
Создал комплексную систему автоматизации процессов с документацией и базой знаний.

## Навыки

- **Backend разработка**: Python, асинхронное программирование
- **Работа с API**: Telegram, Google Sheets, n8n
- **Базы данных**: SQLite, SQLAlchemy
- **DevOps**: Развертывание на Linux серверах
- **Документация**: Техническая документация, структурирование знаний
