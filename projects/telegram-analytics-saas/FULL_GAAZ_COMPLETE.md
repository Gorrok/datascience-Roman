## Как пользоваться

### 1. Регистрация
- Откройте http://localhost:3000
- Нажмите "Начать бесплатно"
- Заполните форму (имя, email, пароль)
- Автоматический вход

### 2. Добавить бота
- После регистрации увидите онбординг
- Создайте бота в @BotFather
- Скопируйте токен
- Вставьте в форму

### 3. Добавить канал
- Перейдите в "Каналы"
- Добавьте бота в канал как админа
- Получите ID канала через @username_to_id_bot
- Добавьте канал

### 4. Смотрите аналитику!
- Dashboard покажет KPI
- График роста
- Топ инвайт-ссылок
- Вся статистика в реальном времени
