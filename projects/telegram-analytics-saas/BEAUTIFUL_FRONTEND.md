# 🎨 Новый Frontend

Я создал красивый, минималистичный интерфейс с фокусом на UX!

## Что изменилось

### ❌ Было (Swagger UI)
- Техническая документация API
- Множество полей и кнопок
- Неинтуитивно для пользователей
- Только для разработчиков

### ✅ Стало
- **Главная страница** с красивым Hero и описанием в 3 шага
- **Простая регистрация** (имя, email, пароль)
- **Онбординг** для новых пользователей
- **Минималистичный дашборд** с KPI и графиками
- **Интуитивно понятный** интерфейс

## Запуск

```bash
# 1. Установите зависимости
cd frontend
npm install

# 2. Создайте .env.local
echo "NEXT_PUBLIC_API_URL=http://localhost:8000" > .env.local

# 3. Запустите
npm run dev

# Откройте http://localhost:3000
```

## Основные экраны

### 1. Главная (`/`)
- Hero с градиентом и CTA
- "Как это работает" в 3 шага
- Карточки фич
- Призыв к действию

### 2. Регистрация (`/auth/register`)
- Простая форма
- Автоматический логин
- Редирект на дашборд

### 3. Логин (`/auth/login`)
- Email + пароль
- JWT токены
- Редирект на дашборд

### 4. Дашборд (`/dashboard`)
- **Онбординг**: если нет ботов — показываем гайд по добавлению
- **KPI**: 4 карточки (всего, активных, новых, ушедших)
- **Графики**: рост канала
- **Топ ссылок**: какие инвайты работают лучше

## Дизайн принципы

✨ **Минимализм**
