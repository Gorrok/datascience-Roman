# 🎨 Запуск Frontend

Красивый, минималистичный интерфейс для Telegram Analytics.

## Быстрый старт

### 1. Установите зависимости

```bash
cd frontend
npm install
```

### 2. Создайте `.env.local`

```bash
# API Backend URL
NEXT_PUBLIC_API_URL=http://localhost:8000
```

### 3. Запустите dev-сервер

```bash
npm run dev
```

Откройте **http://localhost:3000** 🎉

## Что внутри

### Главная страница (`/`)
- Красивый Hero с градиентами
- Раздел "Как это работает" (3 шага)
- Карточки фич
- CTA секции

### Регистрация (`/auth/register`)
- Простая форма (имя, email, пароль)
- Автоматический логин после регистрации
- Редирект на дашборд

### Логин (`/auth/login`)
- Email + пароль
- Сохранение JWT токенов
- Редирект на дашборд

### Дашборд (`/dashboard`)
- **Онбординг для новых пользователей** (добавить первого бота)
- KPI карточки (всего участников, активных, новых, ушедших)
- График роста
- Топ инвайт-ссылок

## Дизайн

### Цвета
- **Primary**: Синий (`#3B82F6`)
- **Secondary**: Фиолетовый (`#8B5CF6`)
- **Градиенты**: `from-blue-50 to-purple-50`
- **Текст**: Тёмно-серый для основного, светло-серый для вторичного

### Стиль
- Минимализм
- Большие интерактивные элементы
- Закруглённые углы (`rounded-xl`)
- Тени для depth (`shadow-lg`, `shadow-xl`)
- Плавные переходы (`transition-all`)

### UI компоненты
- **shadcn/ui** для базовых элементов
- **Lucide Icons** для иконок
- **TailwindCSS** для стилизации
