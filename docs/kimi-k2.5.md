# Kimi AI K2.5 Documentation

Kimi K2.5 — это флагманская мультимодальная модель от Moonshot AI, представляющая собой значительный шаг к сильному ИИ (AGI).

## Основные характеристики

- **Архитектура:** Mixture-of-Experts (MoE).
- **Параметры:** 1 триллион параметров (32 миллиарда активных).
- **Окно контекста:** 256 000 токенов.
- **Обучение:** 15 триллионов смешанных токенов (текст + визуальные данные).

## Ключевые возможности

### 1. Нативная мультимодальность (Vision)
- Преобразование изображений и видео в код.
- Высокая точность в фронтенд-разработке.
- Способность воссоздавать сайты на основе видеозаписей их работы.

### 2. Agent Swarm (Рой агентов)
- Использование технологии Parallel-Agent Reinforcement Learning (PARL).
- Оркестрация до 100 параллельно работающих субагентов.
- Ускорение выполнения сложных задач в 4.5 раза по сравнению с одиночным агентом.

### 3. Офисная продуктивность
- Продвинутая работа с Word, Excel и PDF.
- Обработка документов объемом более 100 страниц.
- Генерация профессиональных отчетов, таблиц и презентаций.

## Доступность
Модель доступна через Kimi Web, мобильное приложение, API и специализированный Kimi Code. Предлагает четыре режима работы: Instant (Мгновенный), Thinking (Размышление), Agent (Агент) и Agent Swarm (Рой агентов).
