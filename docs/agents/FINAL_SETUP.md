```

   → Видит: используется SQLAlchemy

3. Grep "email.*validation"
   → Находит: в schemas/ используют EmailStr из pydantic

4. Применяет тот же паттерн:

   ✅ Следует существующему стилю
   ✅ Не дублирует код
```
