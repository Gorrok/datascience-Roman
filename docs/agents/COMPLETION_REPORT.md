## 🎯 Ключевые особенности

### Автоматическая активация агентов
- `.py` файлы → **Backend Agent**
- `.tsx`, `.jsx`, `.js` → **Frontend Agent**
- `.md` файлы → **Documentation Agent**

### Явный вызов
```
@teamlead <задача>
@backend <задача>
@frontend <задача>
...
```

### Интеллектуальная координация
- TeamLead разбивает сложные задачи
- Агенты работают параллельно где возможно
- Цепочки агентов для последовательных задач
