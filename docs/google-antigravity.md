# Google Antigravity Documentation

Google Antigravity — это платформа для управления ИИ-агентами, состоящая из диспетчера агентов (Agent Manager) и редактора (Editor).

## Начало работы

### Загрузка
Посетите [antigravity.google/download](https://antigravity.google/download) для загрузки.

### Системные требования
- **Linux:** glibc >= 2.28 (например, Ubuntu 20, Debian 10).
- **Windows:** Windows 10 (64-bit).
- **macOS:** Версия 12 (Monterey) и выше. Только процессоры Apple Silicon (X86 не поддерживается).

### Основная навигация
- **Диспетчер агентов:** Можно открыть из редактора кнопкой на верхней панели или горячей клавишей `Cmd + E`.
- **Редактор:** Можно открыть из диспетчера агентов через меню любого рабочего пространства ("Focus Editor") или клавишей `Cmd + E`.
