## Что изменилось

### До

- ❌ Множество полей и кнопок

## Основные экраны

### 1. Главная (`/`)
**Элементы:**
- Hero с градиентом и призывом к действию
- Badge "Запустите за 2 минуты" с анимацией
- Раздел "Как это работает" (3 шага)
- Карточки фич
- CTA секции

**Дизайн:**
- Градиент: `from-blue-50 via-white to-purple-50`
- Крупные заголовки (6xl, 7xl)
- Закруглённые углы (`rounded-xl`)
- Тени для depth

### 2. Регистрация (`/auth/register`)
**Элементы:**
- Простая форма: имя, email, пароль
- Автоматический логин после регистрации
- Ссылка на логин

**UX:**
- Минимум полей (только необходимое)
- Валидация на фронте
- JWT токены сохраняются в localStorage
- Редирект на дашборд

### 3. Логин (`/auth/login`)
**Элементы:**
- Email + пароль
- Обработка ошибок
- Ссылка на регистрацию

### 4. Дашборд (`/dashboard`)

**Онбординг (если ботов нет):**
- Крупная карточка с гайдом
- 3 шага: создать бота → получить токен → вставить токен
- Визуальное разделение шагов (цветные блоки)

**Основной вид (если боты есть):**
- Header с лого и кнопкой настроек
- 4 KPI карточки:
  - Всего участников
  - Активных (зелёный)
  - Новых сегодня (синий)
  - Ушло сегодня (красный)
- График роста
- Топ инвайт-ссылок

## Дизайн система

### Цвета
```css
Primary: #3B82F6 (blue-600)
Secondary: #8B5CF6 (purple-600)
Success: #10B981 (green-600)
Danger: #EF4444 (red-600)
```

### Градиенты
```css
Background: from-blue-50 via-white to-purple-50
Button: from-blue-600 to-purple-600
Text: from-gray-900 via-blue-800 to-purple-900
```

### Типографика
- Заголовки: `font-bold`, размеры 2xl-7xl
- Текст: `text-gray-600` для вторичного
- Антиалиасинг: `antialiased`

### Компоненты
- shadcn/ui для базы (Button, Card)
- Закруглённые углы: `rounded-xl`
- Тени: `shadow-lg`, `shadow-xl`
- Переходы: `transition-all`
- Hover эффекты на всех интерактивных элементах

## Технологии

### Frontend
- Next.js 14 (App Router)
- React 18
- TypeScript
- TailwindCSS
- shadcn/ui
- Axios для API

### API интеграция
- `frontend/lib/api.ts` - централизованный API клиент
- JWT перехватчики для авторизации
- Автоматический refresh токенов
- Обработка ошибок

## Запуск

### Backend (Терминал 1)
```bash
cd backend
source venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### Frontend (Терминал 2)
```bash
cd frontend
npm install
npm run dev
```

Откройте **http://localhost:3000**
